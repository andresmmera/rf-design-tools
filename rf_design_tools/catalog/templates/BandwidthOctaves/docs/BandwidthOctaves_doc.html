Bandwidth in Octaves

BW Octaves Tool

The octave scale indicates how many times the frequency is doubled. It is calculated as follows:

$$N_{octaves} = { {log_2 \left( f_{2} \over f_{1} \right)} }$$

where f2 > f1. The quality factor is defined as:

$${ Q} = {f_c \over BW}$$

Provided that

$$f_c = f_{1} + { BW \over 2} = f_{1} + { {f_{2} - f_{1} } \over 2}$$

Then,

$${ Q} = {f_c \over BW} = { {f_1 + f_2} \over {BW} }$$
